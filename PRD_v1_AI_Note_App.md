# PRD v1 – 온디바이스 AI 지식 저장 앱

## 1. 제품 개요
### 제품 정의
링크나 텍스트를 저장하면 온디바이스 AI가 자동으로 제목과 요약을 생성하고 정리해주는 모바일 앱

### 핵심 가치
- 빠른 저장
- 즉시 이해
- 쉬운 재사용
- 공유 가능

---

## 2. 목표
### 사용자 문제
- 정보가 계속 쌓이지만 휘발됨
- 저장해도 다시 보지 않음
- 정리가 귀찮음

### 해결 방식
- 공유/붙여넣기로 저장
- AI 요약 생성
- 폴더 기반 정리
- 공유 기능 제공

---

## 3. 타겟 사용자
- 정보 저장 많이 하는 사용자
- 육아, 레시피, 쇼핑 관심 사용자
- 콘텐츠 수집 및 재사용 사용자

---

## 4. 핵심 기능
### 저장
- URL 입력
- 텍스트 입력
- 공유 기능

### AI 처리
- 제목 생성
- 3줄 요약
- 대표 이미지 선택

### 폴더
- 개인/공유 폴더
- 편집 권한

### 검색
- 제목/요약/본문 기반 검색

### 오프라인
- 저장/조회/AI 처리 가능
- 온라인 시 동기화

---

## 5. UX 흐름
입력 → 저장 → AI 처리 → 폴더 선택 → 완료

---

## 6. 데이터 모델 (요약)
### Item
- id
- type
- title
- summary
- content
- url
- thumbnail
- folder_id

### Folder
- id
- name
- type
- owner_id

## 7. 기술 스택
- React Native + Expo
- SQLite (로컬 DB)
- Zustand (상태관리)
- Supabase (백엔드)
- 온디바이스 AI (네이티브 모듈)

---

## 8. 아키텍처
